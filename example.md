# 基本機能とスタイル

このスライドではテーマの基本スタイルをテストします。

- **太字** / *斜体* / `code`
- 入れ子リスト
  - 第二階層
  - もう一つ

```ts
function hello(name: string) {
  console.log(`Hello, ${name}`)
}
```

> ヒント: 最初の h1 は固定ヘッダになります。

# フッターとショートカット

このテーマでは Enter / Backspace キーで進行を制御できます。

- Enter: 次のスライド / v-click
- Backspace: 前へ戻る
- フッター: (cover / image 以外) 日付 + ページ番号
